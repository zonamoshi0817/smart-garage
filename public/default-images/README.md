# デフォルト画像フォルダ

このフォルダには、車両のデフォルト画像を配置します。

## 必要な画像ファイル

以下の画像ファイルを配置してください：

- `sedan-blue.jpg` - 青いセダンタイプの車両
- `sedan-silver.jpg` - シルバーのセダンタイプの車両
- `suv-black.jpg` - 黒いSUVタイプの車両
- `sports-red.jpg` - 赤いスポーツカータイプの車両
- `hatchback-white.jpg` - 白いハッチバックタイプの車両
- `truck-gray.jpg` - グレーのトラックタイプの車両

## 画像の仕様

- 形式: JPG
- 推奨サイズ: 400x300px 以上
- ファイルサイズ: 100KB以下推奨
- アスペクト比: 4:3 または 16:9

## 使用方法

1. 上記の画像ファイルをこのフォルダに配置
2. アプリケーションで「デフォルト画像」ボタンから選択可能
3. カテゴリ別に整理されて表示される

## 注意事項

- 著作権フリーの画像を使用してください
- 商用利用可能な画像を選択してください
- 画像の品質は統一感を保つようにしてください
